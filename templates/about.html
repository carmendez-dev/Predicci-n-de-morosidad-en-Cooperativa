Objetivo del Sistema

El Sistema de Predicción de Morosidad de Ahorro Valle utiliza técnicas avanzadas de Machine Learning para evaluar el riesgo crediticio de los solicitantes de préstamos. El sistema analiza múltiples variables para predecir la probabilidad de morosidad en un período de 12 meses.

Modelo de Machine Learning

Modelo Activo: {{ modelo_nombre }}

Algoritmo: Regresión Logística (Optimizada)

Precisión: ~61.8% en conjunto de prueba

Recall (Detección de Morosos): ~56.4%

Specificity (Identificación de Buenos Clientes): ~62.9%

Datos Personales

  • Zona de residencia

Información Laboral

  • Tipo de empleo
  • Antigüedad laboral

Historial Crediticio

Crédito Solicitado

Garantías

Variables Económicas

  • Precio del vino

Proceso de Predicción

1

Recolección de Datos

El usuario ingresa la información del solicitante a través del formulario web.

2

Preprocesamiento

Los datos son validados, normalizados y transformados según el pipeline del modelo.

3

Predicción

El modelo entrenado procesa los datos y genera una predicción de morosidad.

4

Resultado y Recomendación

Se muestra la predicción con probabilidades, nivel de riesgo y recomendaciones.

Clasificación de Riesgo

Riesgo BAJO

Probabilidad de morosidad < 20%

Riesgo MEDIO

Probabilidad de morosidad 20% - 50%

Riesgo ALTO

Probabilidad de morosidad 50% - 70%

Riesgo MUY ALTO

Probabilidad de morosidad > 70%

Consideraciones Importantes

  • Este sistema es una herramienta de apoyo a la decisión, no sustituye el criterio humano.
  • Las predicciones tienen un margen de error y deben considerarse junto con otros factores.
  • Se recomienda realizar evaluaciones adicionales para casos de alto monto o riesgo.
  • El modelo debe ser reentrenado periódicamente con datos actualizados.

Información Técnica

Preprocesamiento: StandardScaler, OneHotEncoder
